Procedural Graphics

Mathematics as Art: a collection of digital assets created algorithmically. The projects are written in javascript using WebGL and the Three.js library

Zionical

Abstracted and colorful environment with interactive and real-time effects, which are created with particles, noise functions, splines, and textures.

Lava Lamp

To visualize the metaballs, the marching cubes algorithm distributes triangles based on the density function of the isosurface in each voxel.

BioCrowds

Crowd simulation using personal space denoted with markers. The space colonization algorithm insures only one agent can own a marker per frame.

River Town

Grammar rules subdivide the building and add windows, roofs, and doors. Low frequency perlin noise is used to create the terrain.

Shaders

A collection of real-time shaders and post-processing effects on a Mario model: MatCap, Iridescent, Toon, Bloom, Vignette.

Wing Generation

Procedural modelling of a bird wing using distribution functions for variation. The position, size, orientation, and color of the feathers are mathematical defined.

Monte Carlo Path Tracer

Components and Features


  • Sampling

    In order to generate sample points on scene objects without bias, I implemented various sampling functions and calculated their probability density functions.


  • Global Illumination

    Developed a recursive integrator for calculating the Light Transport Equation, and sampling the BSDF of specular and lambertian materials.


  • Direct Lighting

    Developed another integrator for the Light Transport Equation that handles direct light - light rays are created from points sampled on area lights.


  • Fresnel Materials

    Created reflective and transmissive BSDF classes by calculating the fresnel reflection coefficient at a point of intersection.


  • Microfacet Shading Model

    Trowbridge-Reitz and Beckmann distribution, ...


  • Full lighting integration

    russian roullete, power heuristic


  • Acceleration Structure

    Implemented a Bounding Volume Hierarchy to contain objects in the scene and perform ray-intersect tests more efficiently. Subdivision of the scene bounding boxes were performed with a surface area heuristic.


  • Geometry

    Implemented geometry intersection functions for cube, cylinder, spheres, planes, disks, constructive solid geometry and meshes.


  • Lights

    Environment Lighting, spot lights, point light
